VeriSafe

VeriSafe is an all-in-one campus safety and security website designed to provide essential safety features using only frontend technologies. It includes an Emergency Alert Button for instant location sharing, Live Location Sharing for real-time tracking, a list of Emergency Contacts for quick access to help, and a Link & File Spam Checker to detect suspicious content. Built entirely with JavaScript, SafeCampus is a lightweight, user-friendly tool to enhance safety for students and faculty on campus.
